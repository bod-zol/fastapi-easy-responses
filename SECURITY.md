# Security Policy

## Supported Versions

By default, always the latest minor version is supported with security patches.

| Version | Supported          |
| ------- | ------------------ |
| 0.x.y   | :white_check_mark: |
| < 0.x.0 | :x:                |

## Reporting a Vulnerability

If you find a vulnerability, please report it using GitHub's private vulnerability reporting.

Please try to be as specific as possible, describing all the steps and example code to reproduce the security issue.
