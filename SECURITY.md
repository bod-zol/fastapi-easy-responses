# Security Policy

## Supported Versions

By default, always the latest minor version is supported with security patches.

| Version | Supported          |
| ------- | ------------------ |
| 0.x.y   | :white_check_mark: |
| < 0.x.y | :x:                |

## Reporting a Vulnerability

If you find a vulnerability, please report it privately by using the "Report vulnerability" button on the repo's [Security and quality](https://github.com/bod-zol/fastapi-easy-responses/security) page.

Please try to be as specific as possible, describing all the steps and example code to reproduce the security issue.
